A tool to read the metadata of the file.<br>
Available extensions are pdf ,jpg ,png.<br>

## Usage

```shell
cd MetaData
pip install pillow,PyPDF2
pyton MetaData.py
```

## Example

```shell
python MetaData.py -f test.pdf
```
```shell
python MetaData.py -f test.jpg
```
